# Qiankun Vite 子应用文档

欢迎来到基于 Vite 和 Qiankun 的微前端子应用开发文档。

## 项目概述

本项目是一个基于 Vite 构建的微前端子应用示例，使用 Qiankun 作为微前端框架，可以独立运行或作为子应用集成到主应用中。

### 技术栈

- Vue 3 + Vite + TypeScript
- 微前端框架：Qiankun
- 构建工具：Vite
- 包管理：PNPM

## 文档导航

- [开发指南](./development/): 详细介绍子应用的开发流程和配置
- [部署指南](./deployment/): 详细介绍如何将子应用部署到 Vercel

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建项目
pnpm build

# 预览构建结果
pnpm preview
```
