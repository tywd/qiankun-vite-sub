# 部署说明

本子应用支持多种部署方式，您可以根据实际需求选择合适的部署方案。

## 部署方式

- [Vercel 部署](./vercel.md): 详细介绍如何将本子应用部署到 Vercel 平台
- [独立部署](./standalone.md): 详细介绍如何将本子应用独立部署到服务器

## 环境变量配置

为了安全和灵活性，建议将敏感配置信息通过部署平台的环境变量功能进行管理，而不是硬编码在代码中。这样可以避免将敏感信息提交到代码仓库中。

在 Vercel 部署中，需要配置以下环境变量：
- `BASE_PATH`: 子应用的基础路径，必须与主应用中配置的 `activeRule` 一致

## 选择建议

- 如果您希望快速部署并获得良好的用户体验，推荐使用 Vercel 部署
- 如果您有特定的服务器环境要求，可以选择独立部署方案
